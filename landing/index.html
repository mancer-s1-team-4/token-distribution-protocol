FAQ

Common questions

What is token vesting?

Vesting is a way to distribute tokens gradually over time rather than all at once. A contributor might be entitled to 10,000 tokens over two years, receiving a portion each month as they continue contributing. It is standard practice in both traditional equity compensation and token-based organizations.

Do I need to understand blockchain to use this?

No. Themis is designed to feel like a standard web application. You will need a Solana wallet to sign transactions, but the interface does not require you to understand how the underlying protocol works.

What happens when a stream is cancelled?

When you cancel a stream, tokens that have already vested remain with the recipient and can still be claimed. Any tokens that had not yet vested are returned to the wallet that created the stream. The cancellation is instant and settled on-chain.

What is a cliff period?

A cliff is a mandatory waiting period before any tokens begin vesting. For example, a one-year cliff means the recipient receives nothing for the first year, then vesting starts. This is common in contributor agreements to ensure commitment before rewards are distributed.

What tokens does Themis support?

Themis supports any SPL token on Solana. That includes USDC, wrapped SOL, and any custom project token that follows the standard SPL token format.
